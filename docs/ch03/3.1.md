### 3.1.1 위치 정책(Location strategy)

- HashLocationStrategy(해시 기반의 내비게이션 정책)

- PathLocationStrategy(방문 기록 API 기반 내비게이션 정책)
  - `http://mysite.com:8080/products/page/3`
